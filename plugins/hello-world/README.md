# Hello World

A deliberately small Claude Code plugin used to demonstrate the repository's eval convention.

Invoke the skill with a name:

```text
/hello-world:hello Ada Lovelace
```

Or use its default:

```text
/hello-world:hello
```

The plugin is intentionally trivial. The interesting part of this repository is the shared harness under `tools/evals/` and the example suites under `evals/` in this plugin.
